ใน Week อาจารย์จริงได้ให้ Review โจทย์มาผมเลยได้มาทบทวนสิ่งที่จะออกสอบทั้งหมดโดยสิ่งที่ได้จากแบบฝึกหัดก็คือ
การใช้พวก Array Function มากกขึ้นได้เข้าใจ Paramiter บางตัวที่อยู่ข้างในบาง Function และยังได้ฝึก Logic ทบทวน
การใช้สิ่งที่เรียนมาไปทั้งหมด พวก Object , spread operator และก็อื่นๆอีกแล้วก็ได้ฝึกอ่าน Doc ของ JS เพื่อทำแบบฝึกหัดของอาจารย์ด้วยครับ
ข้อ 1. เขียนฟังก์ชัน addProperty ที่เพิ่ม property ใหม่เข้าไปใน object พร้อมค่า value 
ข้อ 2. เขียนฟังก์ชัน mergeObject ที่รวม object สองตัวเข้าด้วยกันเป็น object เดียว
ข้อ 3. เขียนฟังก์ชัน extractAndRename ที่ดึงค่า property จาก object ออกมาและเปลี่ยนชื่อ property นั้น
ข้อ 4. เขียนฟังก์ชัน mergeAndDestructure ที่รวม object สองตัวแล้วใช้ destructuring ดึงค่าบาง property ออกมา
ข้อ 5. เขียนฟังก์ชัน convertToUppercase ที่เปลี่ยนค่าของทุก element ใน array ให้เป็นตัวพิมพ์ใหญ่
ข้อ 6. เขียนฟังก์ชัน removeDuplicateWords ที่ลบคำที่ซ้ำออกจากประโยค โดยไม่สนตัวพิมพ์ใหญ่เล็ก
ข้อ 7. เขียนฟังก์ชัน findValue ที่หาค่าต่ำสุด ค่าสูงสุด ผลรวม และค่าเฉลี่ยจาก array ของตัวเลข
ข้อ 8. เขียนฟังก์ชัน findMax ที่หาค่าที่มากที่สุดจาก array ของตัวเลข
ข้อ 9. เขียนฟังก์ชัน doubleAndFilterEvenNumbers ที่ทำการคูณสองทุกตัวเลขใน array แล้วเลือกเก็บเฉพาะเลขคู่
ข้อ 10. เขียนฟังก์ชัน removeFalsyValues ที่ลบค่า falsy ออกจาก array
ข้อ 11. เขียนฟังก์ชัน filterEvenNumbers ที่กรองเอาเฉพาะเลขคู่จาก array ของตัวเลข
ข้อ 12. เขียนฟังก์ชัน populateArray ที่สร้าง array ตามความยาวที่กำหนดและเติมค่าด้วย callback function
ข้อ 13. เขียนฟังก์ชัน squareNumbers ที่ใช้ map ยกกำลังสองตัวเลขทุกตัวใน array
ข้อ 14. เขียนฟังก์ชัน getFreqOfWords ที่นับความถี่ของคำในประโยค โดยไม่สนตัวพิมพ์ใหญ่เล็ก
ข้อ 15. เขียนฟังก์ชัน processNumbers ที่ใช้ reduce เพื่อหาผลรวมของเลขคู่ทั้งหมดใน array
