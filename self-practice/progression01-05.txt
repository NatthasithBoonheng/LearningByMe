ใน Week01 ผมได้เริ่มทบทวนพื้นฐานของ JavaScript โดยทำไฟล์ LongestWord.js และ basic.js เพื่อฝึกการใช้ฟังก์ชัน อาเรย์ และ syntax พื้นฐาน ผ่านโจทย์ประมาณ 10 ข้อ โดยมี การแปลงค่า Celsius การหาค่าสุด การเช็คเลขคู่ การนับตัวอักษร การหาพาลินโดรม และการสร้างตารางสูตร ประมาณนี้จากการฝึกนี้ทำให้ผมรู้ว่าจำเป็นต้องกลับไปทบทวนพื้นฐานให้แน่นก่อนเพราะว่าผมรู้สึกยังไม่ต่อยแม่นสักเท่าไหร่
ใน Week02 ผมได้เน้นทบทวนการใช้ Array Function และ Object มากขึ้น เช่น การใช้ map, reduce, filter, fill และการเข้าถึง key ของ object รวมถึงการใช้ spread operator เพื่อรวม object เหมือนกับรวม array ผมวางแผนไว้ว่าวีคถัดไปว่าจะลองโจทย์ที่ซับซ้อนขึ้นเพราะเริ่มได้ทบทวนพื้นฐานมากขึ้นแล้ว
ใน Week03 ผมได้ทำโจทย์เกี่ยวกับ callback function ทำให้เริ่มเข้าใจการเรียกใช้งานฟังก์ชันภายในฟังก์ชันมากขึ้น ทั้งได้ฝึกการใช้ array function เพิ่มเติม โจทย์ในวีคนี้เน้นการสร้างฟังก์ชันที่รับฟังก์ชันอื่นเป็น parameter และคืนค่าผลลัพธ์ตามเงื่อนไขแล้วหลังจากนั้นก็เข้าใกล้วันสอบมากกขึ้นผมจึงให้ AI ลองเจนโจทย์จาก Reqiment ของอาจารย์มากให้
ใน Week04 ผมได้ทบทวนเนื้อหาเกี่ยวกับ array methods, object, function และ JS module ซึ่งเป็นหัวข้อที่เกี่ยวข้องกับการสอบ ผมให้ ChatGPT ช่วยสร้างโจทย์ฝึกทบทวนตามระดับง่าย กลาง และท้าทาย เช่น การสร้าง array การใช้ length, map, filter, find การเข้าถึง property ของ object การรวม object และการสร้างฟังก์ชัน callback รวมถึง compose แล้วคิดว่าใน Week สุดท้ายก็น่าจะทบทวนเรื่องที่จะสอบเพิ่ม
ใน Week05 ผมได้ทบทวนโจทย์ที่อาจารย์ให้มาเป็นแแบบฝึกหัด ผมเลยได้ฝึกการใช้ array function ทำความเข้าใจ parameter ของฟังก์ชัน ฝึก logic การใช้ object และ spread operator รวมถึงการอ่าน JS documentation เพื่อทำโจทย์ต่าง ๆ เช่น การเพิ่ม property การรวม object การใช้ destructuring การแปลงตัวอักษร การลบคำซ้ำ การหาค่าต่ำสุด ค่าสูงสุด ผลรวม ค่าเฉลี่ย การ filter, map, reduce และการนับความถี่ของคำทั้งหมดทำเพื่อใช้ในการเตรียมตัวสอบครับ
