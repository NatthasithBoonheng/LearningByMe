LongestWord.js ไฟล์นี้ได้ทบทวนเกี่ยวกับการใช้ คำสั่งใน Java script ได้กลับมาฝึกการใช้ Function array และ syntax ต่างๆจากการบ้านที่อาจารย์ให้มา
โดยพอผมได้ลองทำผมรู้สึกได้ว่าผมต้องกลับไป ทบทวนพื้นฐาน ก่อนผมจึงได้มีไฟล์ basic.js มาเพื่อที่จะทบทวนโจทย์ปัญหาพื้นฐานโดยให้ Chatgpt ลอง Gen โจทย์มาให้
ผมได้ลองทบทวนไป 10 ข้อโดยใช้พื้นฐานของ Function เป็นส่วนใหญ่ผมเลยคิดว่า Week 2 จะฝึก Array กับ Object ครับ
ข้อ 1. เขียนฟังก์ชัน celsiusToFahrenheit(celsius) ก็ Return ค่าตามสูตรครับ
ข้อ 2. เขียนฟังค์ชั่นแล้วก็ลูปค่า !n ทั้งหมดครับ
ข้อ 3. เขียนฟังค์ชั่น Return เลขคู่เป็น True Flase ใช้ Condition ปกติครับ
ข้อ 4. แปลง Strng เป็น Array ก่อนแล้วนำมากลับหลังแล้วค่อย Return แบบกลับหลัง
ข้อ 5. หาค่าสุดโดยใช้ ลูป For ลูปเก็บค่าที่มากที่สุดไว้
ข้อ 6. นับจำนวน a,e,i,o,u โดยใช้ลูปแล้วหาตัวอักษรโดยใช้ Condition
ข้อ 7. หาเลขที่หาร 5 กับ 3 ลงตัวโดยใช้ If,else
ข้อ 8. หาพาลินโดรมโดยใช้ Loop For และใช้ Length ดูคำ
ข้อ 9. หาตัวอักษรที่พบบ่อยที่สุดโดยใช้ Loop ไปหาตัวอักษรละเก็บค่า จำนวนกับคำไว้ในตัวแปรโดยเงื่อนไขคือมากที่สุด
ข้อ 10. หาตารางสูตรโดยใช้ Loop For ไล่คูณตั้งแต่ 1 ถึง 12 กับ n ที่รับมา
