จากที่ได้ลองวางแผนจาก Week 1 ว่า Week ที่ 2 จะทบทวนวิธีการใช้ Array Function และ Object ก็พอได้ทบทวนเยอะเลยครับได้เข้าใจวิธีการใช้ Spread Oparator 
... อันนี้มากขึ้นและก็ได้ทบทวน Array Function พวก Map Reduce Fillter อะไรงี้ครับส่วน Object ได้รู้วิธีการเรียก Key Object มาใช้และก็การใช้ 
Spread Oparator ก็ได้พึ่งรู้ว่าเอา มาใช้ Merge Object เหมือน Array ได้ Week หน้าคาดว่าอยากลองอะไรที่ แอดแวนส์ กว่านี้หน่อยครับเลยหลังจากที่ได้เรียนคาบล่าสุดไป
ก็เลยอยากลองฝึกโจทย์คล้ายๆกันดูครับก็คือโจทย์เกี่ยวกับการ สร้าง Function ที่รับ Function กับ เรื่องของ Call Back Function
ไฟล์ Object.js
ข้อ 1. เขียนฟังก์ชัน return ค่า name ของ object นักเรียนครับ
ข้อ 2. เขียนฟังก์ชันเพิ่ม property ใหม่เข้าไปใน object แล้ว return object ที่มี property นั้นครับ
ข้อ 3. เขียนฟังก์ชันนับจำนวน property ใน object โดยใช้ Object.keys() ครับ
ข้อ 4. เขียนฟังก์ชันรวม object สองตัวเข้าด้วยกัน ถ้า key ซ้ำให้ใช้ค่าของ obj2 ครับ
ข้อ 5. เขียนฟังก์ชันคืนค่า array ของ key ทั้งหมดใน object ครับ
ข้อ 6. เขียนฟังก์ชัน list ชื่อ key ทั้งหมดใน object ออกมาเป็น array ครับ
ไฟล์ Array.js
ข้อ 1. เขียนฟังก์ชันหาผลรวมของสมาชิกใน array โดยใช้ reduce() ครับ
ข้อ 2. เขียนฟังก์ชันหาค่าเฉลี่ยของ array โดยใช้ผลรวม/จำนวนสมาชิกครับ
ข้อ 3. เขียนฟังก์ชันลบค่าที่ซ้ำออกจาก array โดยใช้ loop เช็คแล้ว splice ค่าที่ซ้ำออกครับ
ข้อ 4. เขียนฟังก์ชันรวม array สองอันเข้าด้วยกันแล้ว return ค่าใหม่ครับ
ข้อ 5. เขียนฟังก์ชัน reverse array โดยวนลูปจากหลังมาหน้าแล้ว push เข้า array ใหม่ครับ
