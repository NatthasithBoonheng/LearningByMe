วีคนี้ผมได้ลองทำโจทย์ Callback Function แล้วผมก็เริ่มเข้าใจการทำงานของคอลแบคฟังค์ชั่นมากขึ้นและก็ได้เข้าใจการเรียก Function ใน Function อีกทีมากขึ้น
และก็ยังได้ฝึกใช้พวกของ Array Function มากขึ้นโดยเป้าหมายถัดไปผมก็อยากได้ก็อยากหาโจทย์ทบทวนภาพร่วมครับ แล้วอีกไม่กี่วีคก็ใกล้จะสอบผมเลยจะฝึกเนื้อที่เกี่ยวกับการสอบ
ที่อยู่ใน Outline การสอบใน Team ครับ
ข้อ 1. เขียนฟังก์ชัน applyOperation รับเลขสองตัวกับฟังก์ชัน operation แล้ว return ผลจากการเอาเลขสองตัวไปเข้า operation ครับ
ข้อ 2. เขียนฟังก์ชัน filterArray ใช้ callback กรองค่าใน array แล้วคืน array ใหม่ที่ผ่านเงื่อนไขครับ
ข้อ 3. เขียนฟังก์ชัน doTwice รับฟังก์ชันมาแล้วเรียกทำงานซ้ำ 2 รอบครับ
ข้อ 4. เขียนฟังก์ชัน mapArray ใช้ callback แปลงค่าของแต่ละ element ใน array แล้วคืน array ใหม่ครับ
ข้อ 5. เขียนฟังก์ชัน compose รับฟังก์ชัน f, g แล้วทำงาน g ก่อน แล้วเอาผลไปเข้า f ครับ
